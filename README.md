此时拐弯处又传来熟习的乐律，那是我听过不领会几何遍的歌曲《傍晚》。谁人桥畔见证了咱们的故事，谁人店面东家老是重复播放《傍晚》。远在天边偶遇心腹，有年此后回顾起来，感触本人是倒霉儿。
